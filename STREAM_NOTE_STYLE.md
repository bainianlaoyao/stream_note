# Stream Note 简约毛玻璃风格设计

## 设计核心理念

**「界面如玻璃，文字浮其上」**

Stream Note 的视觉风格追求极致简约——没有多余的装饰，毛玻璃不是炫技，而是创造视觉层级。侧边栏像一块磨砂玻璃悬浮在深色背景上，内容区域通透干净。交互如水流般自然，动画符合物理直觉。

---

## 视觉体系

### 色彩系统

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主背景 | `#0D0D0D` | 深邃的黑色，像深夜的书桌 |
| 侧边栏背景 | `rgba(30,30,30,0.6)` + `blur(20px)` | 磨砂玻璃效果 |
| 文字主色 | `#E8E8E8` | 柔和的白色，不刺眼 |
| 文字次要 | `#9A9A9A` | 灰色，用于时间戳等 |
| 强调色 | `#FF9500` | 暖橙色，仅用于关键操作 |

### 字体双模式

- **记录模式（Stream）**：衬线体（Noto Serif SC），像在笔记本上书写
- **工作模式（Views）**：无衬线体（Inter），高效清爽

### 毛玻璃三层级

```
L1 基底层    → #0D0D0D 无模糊
L2 容器层    → rgba(30,30,30,0.6) + blur(20px)  侧边栏/卡片
L3 悬浮层    → rgba(40,40,40,0.8) + blur(30px)   角标/浮层
```

## 关键组件样式

### 1. 侧边栏

- 宽度：收起 60px → 展开 200px
- 毛玻璃背景 + 右侧细边框
- 悬停时微微亮起
- 选中项底部细线指示

### 2. 输入框

- 底部固定，宽度 80%
- 毛玻璃容器
- 聚焦时底边发光的强调色
- 发送按钮悬停变橙色

### 3. 笔记条目

- 极简无边框，只有间距
- 完成项显示删除线 + 变灰
- AI 检测内容：极其微妙的虚线下划线提示

### 4. 待办条目

- 左侧方形复选框
- 完成时对勾弹跳动画
- 右侧溯源箭头仅 12px，悬停时显现

## 动效规范

| 动效 | 时长 | 曲线 |
|------|------|------|
| 侧边栏展开 | 300ms | `cubic-bezier(0.16,1,0.3,1)` |
| 视图切换 | 300ms | 淡入淡出 |
| 按钮点击 | 150ms | 缩小反馈 |
| 笔记淡入 | 400ms | 依次错开 + 上浮 |
